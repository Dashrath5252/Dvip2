# Image Processing Setup

This repository automatically generates responsive images for the Dvip2 website.

## Required Source Images

Place these files in the root directory:

1. `dvip1.webp`
2. `dvip2.webp`
3. `dvip3.webp`
4. `dvip4.webp`
5. `dvipicon512.webp`
